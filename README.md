## Управление

| Клавиши | Действие                                           |
|---------|----------------------------------------------------|
| a       | Анимация                                           |
| s       | Скорость змейки                                    |
| m       | Замедление скорости змейки, при длинее более чем 5 |
| f       | След змейки                                        |
| v       | Векторы                                            |
| g       | Сетка                                              |
| i       | Информационная панель                              |
